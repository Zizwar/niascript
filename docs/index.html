üîß Plugin System API

Complete reference for NiaScript's revolutionary plugin architecture and smart capabilities

Plugin Management

installPlugin(plugin)

Install and validate a new plugin in the system

uninstallPlugin(name)

Remove a plugin from the system

listPlugins()

Get list of all installed plugins

generatePlugin(description)

Generate plugin automatically from description

Smart Processing

processIntent(query, values)

Process natural language with plugin intelligence

extractEntities(text)

Extract entities for translation, email, etc.

findMatchingPlugin(query)

Find best plugin match for user query

executeMCPTools(operation)

Execute Model Context Protocol operations

Plugin Environment Configuration 

# Core AI API (Required)
OPENAI_API_KEY=your_openai_key_here

# Translation Plugin
LIBRETRANSLATE_API_URL=https://libretranslate.com/
GOOGLE_TRANSLATE_API_KEY=your_google_translate_key_here

# Email Plugin
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password
SENDGRID_API_KEY=your_sendgrid_key_here

# MCP Bridge
WEATHER_API_KEY=your_weather_key_here
SEARCH_API_KEY=your_search_key_here

# Plugin System Settings
NIA_PLUGIN_PATH=./plugins/
NIA_LOG_LEVEL=info
NIA_MEMORY_SIZE=1000
